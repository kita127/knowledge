# bash

## コマンド

* 前にいたディレクトリへバック

    cd -

* カレントディレクトリのパスをクリップボードにコピー

    pwd | pbcopy

* pbcopyのでディレクトリ移動

    cd \`pbpaste\`

* 実行時間計測

    time コマンド

* インストールしたパッケージの検索

    dpkg -l

* check directory capacity

    du -s

* zip 解凍

    unzip <file>
* プロンプト(先頭の文字列)変更
    * 環境変数 `$PS1` を変更する

## ショートカットキー

* コマンドの履歴

    Ctrl+r
